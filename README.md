# Arena 3D

原创 3D 竞技场 MOBA 原型，包含六位英雄、兵线、防御塔、技能、移动端触控操作与联机房间大厅。

## 在线试玩

https://yzouj0031-hub.github.io/arena-3d/

## 操作

- 电脑：WASD 移动，鼠标或技能按钮释放技能
- 手机/平板：左侧摇杆移动，右侧技能按钮战斗

角色模型来源与许可见 [LICENSE-ASSETS.txt](./LICENSE-ASSETS.txt)。


## 联机房间

在英雄选择页切换到“联机房间”，房主创建六位房间码并发送给队友；队友输入房间码加入。大厅会显示队友加入/英雄锁定状态，房主可在队友加入后同步开局。

联机大厅使用浏览器原生 `BroadcastChannel`，因此无需注册或第三方服务，但目前仅支持**同一浏览器配置文件下打开同一站点的多个标签页/窗口**（适合本机双窗口联机演示）。跨设备互联网实时对战仍需要部署权威游戏服务器和网络状态同步。


## 跨设备联机测试

打开 https://yzouj0031-hub.github.io/arena-3d/online/ 。该测试场使用现有 Supabase Realtime，支持不同手机、平板和电脑通过六位房间码加入同一房间，最多六人，并同步英雄选择、准备、开局、移动与攻击动作。

当前属于联网链路验证阶段；正式技能伤害和胜负判定仍需后续权威对局逻辑。
